vamos falar sobre interfaces. Uma interface contém definições para um grupo de funcionalidades 
relacionadas que uma classe não abstrata ou uma struct deve implementar.
Usando interfaces, você pode, por exemplo, incluir o comportamento de várias fontes em uma classe.
 Essa funcionalidade é importante em C# porque a linguagem não dá suporte a várias heranças de classes. 
Além disso, use uma interface se você deseja simular a herança para structs, 
pois eles não podem herdar de outro struct ou classe.
